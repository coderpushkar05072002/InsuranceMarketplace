# Insurance Marketplace (Web3)

A decentralized Insurance Marketplace built using **Solidity**, **Hardhat**, **Ethers.js**, and a modern **React + Vite frontend**.

Users can buy insurance policies, submit claims, and interact with the contract using a clean frontend UI.
